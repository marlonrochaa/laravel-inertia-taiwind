# LaVue
> Laravel + Vue.js + Inertia.js + Tailwind CSS

## Features
* Laravel 9
* Vue.js
* Inertia.js
* Tailwind CSS 

## Install
1. clone repo
2. `composer install`
3. `npm install`
4. `cp .env.example .env`
5. `php artisan key:generate`

## Development
1. `npm run dev`
2. open http://localhost:8000/
